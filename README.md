# Event calendar
> An application that supports SCADA system


## General info
This version includes among others:
* Aquiring data from MS SQL.
* Picking data using data pickers.
* Saving selected records to .csv.


## Technologies
* ASP.NET MVC
* MS SQL 2014
* Bootstrap
* Javascript datapicker script
